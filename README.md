# Student Service Portal

A portfolio management web application built using Flask.

## Features

* User Registration & Login
* Dashboard
* Profile Management
* Resume Upload
* Certificate Upload
* Skills Management
* Education Management
* Project Management
* Feedback System
* Admin Panel
* Settings Page

## Tech Stack

* Python
* Flask
* SQLite
* SQLAlchemy
* HTML
* CSS
* Bootstrap

## Run Project

```bash
pip install -r requirements.txt
python app.py
```
